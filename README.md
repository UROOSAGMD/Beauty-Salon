 # Glow Beauty Salon

A modern and responsive Beauty Salon website built with React and Vite. This project features a clean design, smooth animations, and an elegant user experience.

## Features

* Responsive design for all devices
* Hero section with image slider
* About and Services sections
* Beauty Gallery with lightbox
* Client Testimonials
* Booking form with confirmation popup
* Contact section and Newsletter
* Back to Top button
* Smooth scrolling and scroll animations

## Technologies Used

* React
* Vite
* CSS3
* JavaScript (ES6)
* Font Awesome
* Google Fonts

## Installation

Clone the repository and install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build the project:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
